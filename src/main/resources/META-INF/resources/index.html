Όλος ο κώδικας μπορεί να βρεθεί στο https://github.com/gbourant/axinar

Η εφαρμογή έχει γίνει deploy σε AlmaLinux 10 σε VPS της Hetzner

Το DNS έχει ρυθμιστεί με το Cloudflare το οποίο λειτουργεί σαν reverse proxy

Στο VPS υπάρχει ο CaddyServer όπου έχει εκδώσει δικό του certificate και λειτουργεί και αυτός σαν reverse proxy

Το systemd έχει χρησιμοποιηθεί ώστε να ξεκινά την εφαρμογή μετά από boot και το network ή failure

Έχουν γίνει και τα 3 coding challenges:

  1. Η πρώτη εφαρμογή είναι live στο REST Endpoint, και όλα τα crud operations είναι live και διαθέσιμα για να τα δοκιμάσετε. Επίσης, υπάρχει διαθέσιμο Swagger UI και ο κώδικας μπορεί να βρεθεί εδώ
  2. Η δεύτερη άσκηση είναι διαθέσιμη εδώ reorder.html και ο κώδικας εδώ αλλά και με δεξί κλικ -> view-source
  3. Η τρίτη άσκηση είναι διαθέσιμη εδώ FindCsvDuplicates.java όπως και στο repo εδώ

Για την πρώτη άσκηση έχει χρησιμοποιηθεί η H2 database και ο κώδικας είναι πλήρες τεσταρισμένος
